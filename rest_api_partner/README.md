Usage
=====
* Install some module from oca
  * base_rest
  * base_rest_datamodel
  * base_rest_pydantic
  * component
  * extendable
  * pydantic
* Install module for auth api
  * auth_api_generate_key(https://github.com/kurniawanlucky/Odoo10.0/tree/15.0/auth_api_generate_key)
* Create Api key
  * At Settings > Technical > Auth Api Key
